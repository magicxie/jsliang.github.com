Sheetaholics 筆記內頁產生器 V1.1

by J. S. Liang
in Blog on 2011-04-18 | tags: PyInstaller Python Sheetaholics pywin32 reportlab wxFormBuilder wxPython 技術筆記 文具 | comments

Sheetaholics是可用來產生筆記本內頁的程式。

名字拆開來看為sheet-aholics，「內頁狂」的意思。此為自創詞，請小心實際上並沒有這個單字……XD

Sheetaholics的前身是我之前寫的另一支網頁程式：東大筆記本(點線筆記)內頁產生器。原本的程式讓網友們可以透過網頁的介面產生如下圖的點線筆記本內頁：

▲ 自製點線筆記本內頁：A4尺寸，左邊的方眼是8mm間距，右邊則是6mm間距

內頁產生器推出後比我預想的還要受歡迎，結果這支程式的流量負載每個月花費將近3美金。雖然不算貴，累積個把月的金額也夠買些奢侈品了（例如20美金可以買80GB/年的Google Storage呢），身為想存錢的窮學生，只得將服務暫停……

服務暫停後，還是有不少朋友前來索取PDF檔。

所以為了能讓大家使用更方便（一部份也是因為最近做研究遇到瓶頸，想找個出口喘息一下啊～Orz），我著手把原本的程式改成離線單機版，也就是現在的Sheetaholics。

Sheetaholics的程式介面如下圖所示，設定完成後按右下的「產生PDF檔」鈕即可。

能夠做的設定和之前的內頁產生器是完全一樣的。:)

按下「產生PDF檔」鈕後，PDF檔就會輸出到指定的目錄。之後將PDF檔列印出來即可。

目前我手邊有的環境是Windows 7 x86和Ubuntu 10.04，測試結果程式皆可正常執行。

其他環境還沒試過，如果有使用的網友們能幫忙回報一下各種環境的支援狀況將非常感激。:)

Windows

  1. 下載主程式：sheetaholics-dottedlined-win32-1.1.exe
  2. 下載後，直接點兩下程式即可執行，不需要安裝。

目前確定Windows 7 x86可以正常執行。

補充：Windows 7 x64亦可正常執行。（感謝PTT文具板Anal板友提供資訊）

在XP執行時如果出現「無法找到程序輸入點」訊息，可能是因為程式所在的路徑有中文，例如：「C:...桌面sheetaholicssheetaholics-dottedlined-win32-1.1.exe」（感謝史羅爾以及PTT文具板的Wengboyu板友提供資訊）

解決方法是將程式移到路徑全部是英文的資料夾內執行。最簡單就是直接放在「C:」下了。

如果執行過程中還有其他錯誤，請再告訴我。:)

UNIX-like環境（以Ubuntu為例）

Sheetaholics完全是用Python寫的，理論上直接抓程式碼下來執行即可。 以下安裝步驟以Ubuntu為例。

  1. 安裝所需的Python套件：

    sudo apt-get update sudo apt-get install python-reportlab sudo apt-get install python-wxgtk2.8 python-wxtools wx2.8-i18n

  2. 安裝完畢後，再取得Sheetaholics程式碼並解壓縮：

    wget https://github.com/jsliang/sheetaholics/tarball/Sheetaholics-1.1.0 --no-check-certificate tar xvf Sheetaholics-1.1.0

  3. 解壓縮完即可執行：

    python jsliang-sheetaholics-2a6d746/dottedlined/dottedlined.py

Mac

根據PTT文具板hsiyen網友回報，Windows的執行檔在Mac上可以用Wine可以開啟，但是顏色只能選擇黑色，選擇其他顏色頁面會變全白……Orz 我個人沒有Mac，是否有人可以幫忙試著參考前述UNIX-like環境安裝套件，直接用Python執行看看？或是有人能幫忙建Mac的執行檔嗎？XD
