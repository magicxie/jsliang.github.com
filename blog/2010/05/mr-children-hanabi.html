Mr. Children - HANABI (Code Blue第一季主題曲)

by jsliang
in Blog on 2010-05-23   comments

tags: 戲劇 日本語 歌詞 音樂

Mr. Children的HANABI是日劇《コード・ブルー -ドクターヘリ緊急救命-》（台譯：空中急診英雄）第一季的主題曲。

Code Blue以急救為主題，比較特別的是，主角群的工作是搭直升機去現場進行急救。

乍看之下很酷，但所有醫師會經歷的他們也少不了，必須天天面對生老病死。

希望能救人，卻又希望能遇到嚴重病例、或是期待患者的病情惡化，好增加自己的經驗。與之伴隨而來的各種心理上的矛盾衝突，不斷地衝擊著他們當醫師的決心。

Code Blue整體來說是很感人的日劇，但也非常嚴肅。即使主角群有幾位甘草人物，也完全無法稱上是輕鬆小品。

而在沉重的一集結束後，片尾曲HANABI（煙火）總是有平衡的效果。

雖然我的日文還不夠好到可以明白全部的歌詞，不過至少毎段副歌都要重複四次的「もう一回（再一次）」我還聽得懂。:P

雖然現實令人如此挫敗，不過不要放棄，再努力一次吧！
