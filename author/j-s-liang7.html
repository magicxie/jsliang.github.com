    同時用Mercurial和Git，在branch的使用上感到很錯亂，昨天研究了一下，今天在此做個小小的筆記。

    如果有理解錯誤之處，也麻煩朋友們協助指正，感激不盡。

    Mercurial的named branch的指令操作和Git差不多，但差別在於：

    • Mercurial的branch資訊是存在每個commit資料裡，並且預設無法修改
    • Git的branch資訊與commit資料是分開的，branch比較像是一種特殊的tag

    以下面的範例為例：

    # 產生一個新的hg repository，命名為hg_namedbranches
mkdir hg_namedbranches
cd hg_namedbranches
hg init

# Rev 0: 隨便新增一個檔案並commit
touch a
hg add a
hg commit -m "Commit 1"

# Rev 1: 把a更名為b並commit
hg mv a b
hg commit -m "Commit 2"

  6. 隱喻法

    by J. S. Liang
    in Blog on 2011-06-28 | tags: 心情 | comments

    有點難想像：國中的我愛寫小說，高中的我愛寫新詩。

    雖然那時寫的東西，為賦新辭強說愁的比例很高，但至少有努力把當時從國文課還有課外書中學到的各種修辭法應用進去。

    可惜那時的作品和日記沒有好好留下來，想必文風和現在差很多吧！

    就像前幾天跟阿達聊到的，以前我可能可以自然地寫出像「傾盆而下的大雨是我的悲傷」這樣的隱喻句。

    不過現在我頂多寫明喻句，例如：「我的眼淚就像現在窗外的大雨一樣滂沱而下。」

    更實際點，在發噗或是更新Facebook狀態時，我寫的八成會是：「我哭得好慘，心情已經夠難過了，外面還下大雨……」

    雖然使用隱喻的方式，往往可以營造出某種特殊的美感，然而現在的我已經無法使用隱喻了。

    即使明知道自己不是在寫嚴謹的文件，但當A和B是完全不同的東西時，要寫出「A是B」這類字面上很顯然是錯誤定義的句子，不知怎地心裡就是會感到極度不舒服。

    想不到現在的我的想像力居然變得比中學時候還要膽小放不開了。

    幸運的是，我還是可以理解隱喻修辭法的美，只是認清自己寫不出來。 那也不用勉強，就靜靜地欣賞別人的作品吧。:P

    in Blog on 2011-10-19 | tags: 心情 | comments

    這次醫生開給我的藥，藥名難得地簡短：MgO，氧化鎂。

    我努力回想，才想起中學化學的確有教過氧化鎂是很常見的瀉藥。

    然後接著想起一個遺忘更久的事實：我高中讀理組時，最強的科目其實是化學啊！

    而且我理科最弱的就是數學，弱到我非常懷疑大學是不是不該選資工系。

    如果不是因為那時確定自己愛寫程式，在那對未來茫然不知何去何從的高中年代，我應該會參考自己的成績分佈而選填化學相關科系吧。

    Steve Jobs過世後，他2005年在史丹福大學畢業典禮上的演講開始在社群網路上被廣泛轉載，演講中最被大家關注的重點之一是「不論在哪方面，都要追求自己所愛的人、事、物，別輕易妥協」。

    我覺得自己很幸運，很小的時候就找到一件自己很愛做的事。

    一直都知道自己寫的程式不怎麼樣。操作不比別人寫的好用，效能也不比別人寫的好。（唔，這個句子寫在blog上，以後找工作被想應徵的公司的HR看到會不會因此碰壁啊…… XD）

    但很奇妙的是，即使心裡很清楚這些地方我都不強，我卻不會像在學其他領域時一樣，有強烈的想放棄的挫敗感。

    學跳舞，我常覺得自己怎樣練都無法跳得跟別人一樣好，往往因此挫折到連練習都不想。

    學攝影，我也常覺得自己不論美感和技巧都不如人，因此退縮到「有記錄下想記錄的畫面就夠了」的標準，設備也很乾脆地賣了。

    by J. S. Liang
    in Blog on 2010-05-23 | tags: 戲劇 日本語 歌詞 音樂 | comments

    Mr. Children的HANABI是日劇《コード・ブルー -ドクターヘリ緊急救命-》（台譯：空中急診英雄）第一季的主題曲。

    Code Blue以急救為主題，比較特別的是，主角群的工作是搭直升機去現場進行急救。

    乍看之下很酷，但所有醫師會經歷的他們也少不了，必須天天面對生老病死。

    希望能救人，卻又希望能遇到嚴重病例、或是期待患者的病情惡化，好增加自己的經驗。與之伴隨而來的各種心理上的矛盾衝突，不斷地衝擊著他們當醫師的決心。

    Code Blue整體來說是很感人的日劇，但也非常嚴肅。即使主角群有幾位甘草人物，也完全無法稱上是輕鬆小品。

    而在沉重的一集結束後，片尾曲HANABI（煙火）總是有平衡的效果。

    雖然我的日文還不夠好到可以明白全部的歌詞，不過至少毎段副歌都要重複四次的「もう一回（再一次）」我還聽得懂。:P
